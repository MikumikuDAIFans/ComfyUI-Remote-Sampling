# Target Plan Index

## Target Set

| Target | Name | Outcome |
|---|---|---|
| T1 | Workflow-Level Product Entry | User can plan/run current canvas from a workflow-level control. |
| T2 | Source Truth And Conversion Equivalence | Current local workflow is the only trusted source; stale converted artifacts are rejected. |
| T3 | Resource And Custom Node Alignment | Models, LoRA and custom nodes are mirrored, verified or fail closed. |
| T4 | Observable Remote Execution | Long-running sync/conversion/sampling has live status, events, report and UI evidence. |
| T5 | Privacy, Real Validation And Release | Remote never sees RGB images; real matrix, review, commit and push complete. |

Each target is implemented by the Phase 0-9 task book and must produce evidence under `evidence/`.
